Procedural Dragon Animation

Project Info:
  • Personal Artwork
  • Unity & C# & Houdini & HLSL
  • Procedural Animation / Physically based Simulation

This is a simple implementation of physically based procedural dragon animation. It supports the following functions:

  • procedural rout generation for chasing, diving, gliding.
  • a physically based state machine.

All the showcases below are all implemented in Unity C# in real-time.
